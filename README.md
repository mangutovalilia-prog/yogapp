# Прана 🧘 — йога с тобой

Прототип йога-приложения (пока как сайт). Чистый HTML/CSS/JS, без сборки и зависимостей.

## Как открыть

**Просто:** двойной клик по `index.html` — откроется в браузере.

**Лучше (чтобы работали ссылки-приглашения):** запустить локальный сервер:

```bash
cd ~/йога
python3 -m http.server 8000
```

Затем открыть http://localhost:8000

## Что внутри

| Раздел | Что делает |
|--------|-----------|
| **Асаны** | 3 уровня сложности (🟢🟡🟣). У каждой асаны: персонаж в позе, подсветка мышц, вариации, влияние по ведам и Патанджали, подсказки наставника. |
| **Дыхание** | Практики по состоянию (расслабляющие / балансирующие / бодрящие) с живой анимацией вдоха-выдоха. |
| **Тренировка** | Генератор по времени, интенсивности и состоянию. Добавляет дыхательные практики. Плеер ведёт за руку. Ссылку можно отправить другу с общим временем старта. |
| **Советы** | Цитаты, рекомендации и интересные факты о йоге. |

**Персонаж** (женщина / мужчина / зайчиха / леопард) выбирается кнопкой справа вверху и сохраняется.

## Структура

```
index.html          # каркас и навигация
css/styles.css      # дизайн-система
js/data.js          # весь контент (асаны, дыхание, советы) — легко расширять
js/characters.js    # SVG-персонажи, позы и схема мышц
js/app.js           # роутинг, экраны, генератор, плеер, ссылки
```
